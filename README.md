# PathPointer

PathPointer imports a GPX track or route, finds the closest point on that path, and shows live distance and compass guidance toward it.

Open the hosted app on an iPhone, allow location access, import a GPX file, and tap **Enable compass**. For an app-like experience, use Safari’s **Share → Add to Home Screen**.

GPX files and location data remain in the browser on the device. Nothing is uploaded by the app.
